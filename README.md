# C Algorithm Template Library (CATL)

CATL is a C library offering algorithm templates for learning purposes. The focus is on clarity and simplicity, allowing anyone to explore and understand how common data structures and algorithms are implemented, while also providing code that can be used in projects without worrying about copyright license or production readiness.

## Already implemented:
- array-list (todo: implement possible capacity reduction)
- linked-list
- pair
- bitmasks (1-64 bits positions)
- stack (todo: refactory to array list)
- queue (todo: refactory to use unrolled linked list)
- arena
