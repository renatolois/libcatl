Create a new linked list.

The memory must be freed by catl_linked_list_destroy()

Parameters
delete_functionPointer to function used by catl_linked_list_destroy() and catl_linked_list_delete()

Uses free() if delete_function == NULL

Returns
catl_linked_list_t pointer or NULL if failed to allocate memory

Delete a node from the list.

Parameters

idxIndex of the node that will be deleted from the list
Returns
CatlExitStatus indicating the result of the operation:

  • CATL_STATUS_INVALID_MEMORY if list pointer is NULL
